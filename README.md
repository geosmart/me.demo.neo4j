# 在嵌入式程序中使用Neo4j Browser
1. 下载Neo4j Windows版；  
2. 更改neo4j-server.properties中的org.neo4j.server.database.location属性为嵌入式程序的数据库路径；  
3. 启动Neo4j客户端程序；  
4. 在浏览器打开[Neo4j Browser](http://localhost:7474/browser/)；
